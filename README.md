# Koresuki Support

## About

Slack に これすき な URL を投稿したいときに これすき なチャンネルを開かずとも、 Slash コマンド `/nsfw` を通して Twitter の URL を送信すれば NSFW なチャンネルに投稿してくれるやつ。

NSFW 対応もあるよ！
